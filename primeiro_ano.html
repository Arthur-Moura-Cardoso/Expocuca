1° ANO - ENSINO MÉDIO

LINGUAGEM, TRABALHO E TECNOLOGIA

Linguagem, trabalho e tecnologia - LTT, tem como fins compreender a estrutura e funcionamento da empresa com base nos objetivos dela, auxiliando na rotina dos vários departamentos, estimulando a divisão de trabalho e o trabalho em equipe permitindo a qualidade dos produtos e serviços.

PROGRAMAÇÃO E ALGORITMOS

Os algoritmos são conjuntos de instruções que orientam a execução de tarefas específicas, enquanto a lógica de programação é a maneira como esses algoritmos são organizados para solucionar problemas e atingir um resultado desejado.

BANCO DE DADOS I

O banco de dados é uma ferramenta de armazenamento de dados que possibilita o gerenciamento total das informações coletadas. Por meio do banco de dados é possível analisar qualquer alteração relacionada à empresa, desde seu fluxo de caixa até os dados sobre o público-alvo.

ANÁLISE E PROJETOS DE SISTEMAS

Análise e Desenvolvimento de Sistemas é uma graduação de nível tecnológico que torna o estudante apto a projetar, testar, implantar e gerenciar sistemas e recursos que envolvem Hardware e Software.

FUNDAMENTOS DA INFORMÁTICA

A Informática é um ramo das ciências da informação e da computação. Estuda os processos de recolha, armazenamento, processamento, transferência e difusão de dados digitais. O termo surgiu da junção das palavras informação e automática e pode ser resumido como o processo de tratamento automático da informação.

TÉCNICAS DE PROGRAMAÇÃO

A disciplina de Técnicas de Programação compreende os conceitos e boas práticas para a criação de programas de computadores, através de técnicas e recursos como: variáveis, tipos de dados, modularização, estruturas de con- trole, funções e procedimentos e recursividade.

DESIGN DIGITAL

Os alunos aprendem como criar conteúdo visual para a web, como desenvolver interface de usuário, projetos de desenvolvimento de aplicativos, edição de vídeos, criação de games, design de ícones, desenvolvimento de sites, animação digital entre muito mais.

PROGRAMAÇÃO WEB I

As linguagens de programação web são utilizadas para o desenvolvimentos das camadas de apresentação e de lógica de negócio de web sites. Python, PHP, Go, e Ruby ajudam os bancos de dados a se comunicarem com aplicações no back-end de um programa. Desenvolvimento de aplicações web. C#, JavaScript, Java, Go, e Ruby estão entre as melhores linguagens de programação para construção de web apps.

INGLÊS INSTRUMENTAL

O inglês instrumental é um método de aprendizado que acelera a leitura e a interpretação de textos no idioma e é indicado para pessoas que possuem fins específicos na sua carreira — entre eles, processos de mestrado e doutorado ou concursos.

DESENVOLVIMENTO DE SISTEMAS

Desenvolvimento de sistemas é justamente a área que se ocupa em criar soluções informatizadas, manipulando inteligentemente dados, a fim de que se construam programas, softwares ou aplicações digitais.

BANCO DE DADOS II

A disciplina deve apresentar as técnicas para segurança e integridade de banco de dados; as técnicas para controle de concorrência e recuperação de banco de dados; as novas aplicações de banco de dados e os novos modelos de dados.

INTERNET E PROTOCOLOS

Protocolo de Internet é um protocolo de comunicação usado entre todas as máquinas em rede para encaminhamento dos dados. Tanto no Modelo TCP/IP, quanto no Modelo OSI, o importante protocolo da internet IP está na camada intitulada camada de rede.

PROGRAMAÇÃO DE APLICATIVO

De maneira resumida, as linguagens de programação fazem os aplicativos operarem em um determinado sistema operacional. Elas ditam as “regras” dentro do software em questão, executando ações diversas a partir de uma comunicação estabelecida entre o dispositivo e o app.

PROGRAMAÇÃO WEB II

O desenvolvimento web inclui criação, codificação e programação de sites e seus respectivos elementos e linguagens. Ele também envolve integração com sistemas e ferramentas, armazenamento de banco de dados, cibersegurança, entre outros.

PLANEJAMENTO DO TRABALHO DE CONCLUSÃO DE CURSO (TCC) - TCC

que significa Trabalho de Conclusão de Curso, é uma avaliação feita no final da graduação. Sua finalidade é que o aluno escreva no papel tudo o que aprendeu desde o início de seus estudos. Portanto, é possível saber se tudo está devidamente absorvido e compreendido.
